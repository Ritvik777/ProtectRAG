Open source · PyPI · Python 3.10+

Screen RAG chunks for prompt injection before they reach your model.

ProtectRAG is a lightweight library: fast heuristic rules, optional OpenAI-compatible LLM classification, allow / warn / block policies, structured logs, metrics, callbacks, and integrations for FastAPI, LangChain, and LlamaIndex.

Install from PyPI View on GitHub

What it is

Text in your knowledge base can contain hidden instructions (prompt injection). When a retriever returns that chunk, the assistant may obey those instructions instead of your app policy. ProtectRAG analyzes each chunk’s text and assigns severity using regex-based detectors and/or an LLM, then you decide whether to index it, quarantine it, or drop it at retrieval time.

The library does not connect to vector databases or host an API — you invoke it from your ingestion pipeline, workers, or app server.

How it works

High-level flow from raw text to policy and observability.

flowchart TB
  T[Text chunk + document_id] --> S[Scan]
  S --> H[Heuristics: 13+ rule families]
  S --> L[Optional: LLM classifier]
  S --> Y[Hybrid: heuristics first, LLM when needed]
  H --> P[Policy]
  L --> P
  Y --> P
  P --> A{Decision}
  A -->|below thresholds| AL[ALLOW]
  A -->|medium| W[ALLOW_WITH_WARNING]
  A -->|high| B[BLOCK]
  AL --> O[Logs · Metrics · Callbacks · OTel]
  W --> O
  B --> O

Async & scale

async_scan_batch with concurrency limits and chunking, Redis-backed LLM cache, parallel LangChain/LlamaIndex transforms, and httpx connection pool tuning.

Quality

Golden dataset and run_eval_dataset for regression testing when you change rules or models.

Heuristic detection (sample)

Thirteen documented rule families; the library ships compiled patterns for speed. Examples:

Family Examples
Instruction override “ignore previous instructions”, “you are now…”, “act as if”
Multilingual Chinese, Japanese, French, Spanish, German, Arabic, Korean overrides
Role delimiters <|system|>, [SYSTEM]:, fake role blocks
Exfiltration “send docs to URL”, markdown/HTML exfil patterns
Obfuscation Base64/rot13 hints, zero-width & bidi Unicode
Indirect “When the user asks X, reply Y instead”

See the full table in the README on GitHub.

Install

pip install protectrag                    # core, zero required deps
pip install "protectrag[llm]"             # + httpx for LLM
pip install "protectrag[langchain]"       # LangChain
pip install "protectrag[llamaindex]"      # LlamaIndex
pip install "protectrag[fastapi]"         # FastAPI
pip install "protectrag[otel]"            # OpenTelemetry
pip install "protectrag[redis]"           # shared LLM cache

Quick usage: ingest_document(text, document_id=...) with optional scan=hybrid.scan, or screen_retrieved_chunks on retrieved chunks.
